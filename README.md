# FUTURE_CS_03 – API Security Risk Analysis

## Future Interns Cyber Security Internship

### 📌 Project Overview

This repository contains the work completed for **Task 3 – API Security Risk Analysis** as part of the **Future Interns Cyber Security Internship**.

The objective of this project was to perform a **read-only security assessment** of a public REST API by evaluating authentication mechanisms, HTTP security headers, access control, error handling, CORS configuration, rate limiting, and other common API security practices. The assessment was conducted without exploiting vulnerabilities, following ethical cybersecurity guidelines.

---

## 🎯 Objectives

* Perform a read-only API security assessment.
* Analyze API authentication and authorization.
* Review HTTP response headers.
* Evaluate CORS configuration.
* Check error handling and information disclosure.
* Identify potential security risks.
* Recommend security best practices.
